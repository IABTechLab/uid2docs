UID2 の定義、ID タイプ、指針、構成要素、その他の概念的な詳細については、[UID2 Overview](intro.md) を参照してください。連絡先とライセンス情報、正規化とハッシュエンコーディングルールについては、[Getting Started](/docs/category/getting-started) を参照してください。
