[UID Verify Chrome extension](../ref-info/ref-uid-verify.md) を使用すると、Chrome で直接 UID2 Web インテグレーションを検査およびデバッグできます。UID Verify は、JavaScript 用 SDK、Prebid.js、および Google Secure Signals をサポートしています。
