# Audit Trail

UID2 Portal では、管理ユーザーは **Audit Trail** ページにアクセスして、現在の参加者によって実行された過去のすべてのアクションの詳細なログを表示できます。テーブルには、実行されたアクション、実行したユーザー、実行された時間、および成功したかどうかが表示されます。Audit Trail テーブルは、ソートと検索もサポートしています。
